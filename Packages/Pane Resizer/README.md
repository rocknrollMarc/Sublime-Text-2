Sublime Pane Resizer
======

A Sublime Text 2 plugin that allows panes to be resized without a mouse.


Hotkeys
------

Increase pane size

`alt + =`

Decrease pane size

`alt + -`

Cycle through panes in Windows/Linux:

```
alt + pageup
alt + pagedown
```

OSX:

```
alt + tab
alt + shift + tab
```
